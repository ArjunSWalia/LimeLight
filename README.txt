Initialize npm projects:
  cd frontend
  npm i
  cd ../backend
  npm i

Initialize docker:
  install docker
  docker-compose -f compose.yaml up

Running the project:

  cd backend
  node server.js
  (in a seperate terminal instance)
  cd frontend
  npm run start
  

Dataset source: https://www.kaggle.com/datasets/azathoth42/myanimelist/discussion
